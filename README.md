# 🍳 WhatCanICook (Feeders)

A modern recipe discovery app that helps you decide what to cook based on available ingredients.  
Search recipes, explore ingredients, and get inspired with meal ideas instantly.

---

## 🚀 Features

- 🔍 Search recipes by ingredients
- 🧑‍🍳 Explore full recipe details (instructions, images, videos)
- 🥦 Ingredient-based filtering system
- 📦 Categorized food system (Vegetables, Fruits, Pantry, etc.)
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design
